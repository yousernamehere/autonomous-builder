# Outbound Sprint — Tracking System

This is the minimal tracking schema for the 20-message cadence.

## Daily target
- 10 cold opens
- 5 follow-ups
- 5 warm/community replies

## Tracking
Use `docs/templates/outreach-log.csv` as the schema.

### Status values
- `draft`
- `sent`
- `replied`
- `booked`
- `closed_lost`

## Review loop
- End of day: count sent + replies
- End of week: identify best-performing message + ICP angle
